📌 AI Mock Interview App (Flutter)
An AI-powered mock interview application built using Flutter that helps users practice technical interviews in real-time.

🔹 Choose your field (App, Web, Software Development)

🔹 Select technology (Flutter, React, Java, etc.)

🔹 Generate unique interview questions using AI

🔹 Submit answers and get instant evaluation

🔹 Feedback includes score, mistakes, correct answer, and tips

🚀 Designed to improve interview skills with real-time AI guidance.

🛠 Tech Stack

Flutter (UI)
Dart
REST API (OpenRouter / OpenAI)

🎯 Features

Dynamic question generation
Answer evaluation system
Clean and simple UI
Real-time API integration
